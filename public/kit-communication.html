Installez l'appli sur votre téléphone
Comme une vraie application — sans passer par un store. Gratuit, en 20 secondes.
iPhone (Safari)
  1. Ouvrez le site dans Safari
  2. Touchez l'icône Partager
  3. « Sur l'écran d'accueil »
  4. Confirmez : Ajouter
Android (Chrome)
  1. Ouvrez le site dans Chrome
  2. Menu ⋮ en haut à droite
  3. « Ajouter à l'écran d'accueil »
  4. Confirmez : Installer
